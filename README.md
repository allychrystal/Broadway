# Broadway Capstone

The study of all Broadway shows over the last 12 years.

Comparing the past revenue, attendance and ticket prices per week with the features of that show.

This project predicts which Broadway shows in upcoming 2018 will be successes.  

Only 25% of Broadway shows are successfully profitable- can we predict the next Hamilton?  

Data scraped from Playbill Pro and Internet Broadway Database.

Speciality shows not included.
